## 서론
우리는 다음과 같은 방식으로 Repository를 관리할 예정입니다.
![](repositoryFeature.png)
이런 방식으로 사용하는 이유는 몇가지가 있습니다만, PR을 활용하여 서로에 대한 코드를 리뷰해주는것이 가장 큰 목적입니다.
